◆ start()

void olivera::Core::start ( )

Core while loop.

Record mouse and keyboard inputs and sends them to their respective objects. All entites go through their tick functions, and then through their dispay functions. A viewport is set up using the user inputed window width and height. Further viewports are created for user defined cameras.
